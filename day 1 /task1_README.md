# Day 1 - Profile Card App

## Description

This Flutter application presents a professional profile card using modern UI components and a clean design.

The app displays personal information, role, contact details, and a short professional description. It also includes a button that updates the profile status and shows a SnackBar message.

## Features

* MaterialApp and Scaffold
* Professional profile card layout
* Profile icon/avatar
* Name and role display
* Short professional description
* Contact information
* Row and Column widgets
* Container, Card and Padding widgets
* Interactive button with SnackBar
* Modern cold-color theme

## Technologies

* Flutter
* Dart

## Learning Objectives

* Understanding Flutter widget structure
* Using StatelessWidget and StatefulWidget
* Working with Row and Column layouts
* Styling widgets with colors and spacing
* Handling button interactions
* Displaying SnackBar messages
