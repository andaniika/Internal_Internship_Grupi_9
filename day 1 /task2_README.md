# Day 2 - Grade Calculator App

## Description

This Flutter application calculates the average of three grades entered by the user.

The app validates user input, calculates the average score, and displays a status indicating whether the student passes or needs improvement.

## Features

* Three numeric input fields
* Input validation
* Average calculation
* Display average with two decimal places
* Pass / Needs Improvement status
* Responsive card-based layout
* Clean and user-friendly interface

## Technologies

* Flutter
* Dart

## Learning Objectives

* Working with TextField widgets
* Using TextEditingController
* Input validation
* Using conditional statements (if/else)
* Performing calculations with Dart
* Updating UI with StatefulWidget
* Displaying dynamic results

## Validation Rules

* All fields must be filled
* Only numeric values are accepted
* Invalid input displays an error message

## Pass Criteria

* Average >= 50 → Pass
* Average < 50 → Needs Improvement
