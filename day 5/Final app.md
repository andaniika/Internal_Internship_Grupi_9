# Task Manager App

## Overview

Task Manager App is a Flutter application that helps users manage their daily tasks. Users can create, update, delete, and track the completion status of tasks through a simple and user-friendly interface.

---

## Features

* Add new tasks
* Edit existing tasks
* Delete tasks
* Mark tasks as completed or pending
* View task details on a separate screen
* Input validation for empty task titles
* SnackBar feedback messages
* Modern Material 3 UI design

---

## Technologies Used

* Flutter
* Dart
* Material 3
* StatefulWidget
* setState()
* Navigator.push() / Navigator.pop()

## Functionalities

### Create Task

Users can create a new task by entering a title and description.

### Read Tasks

All tasks are displayed in a scrollable list using ListView.

### Update Task

Users can edit task information at any time.

### Delete Task

Tasks can be removed from the list.

### Task Status

Tasks can be marked as completed or pending using a checkbox.

### Details Screen

Clicking on a task opens a details page showing additional information.
